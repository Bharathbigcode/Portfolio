  • Level Designer

    In level designing, one approach is to use references from other games to create a layout and build it in a 2D view. Another approach is to create a 3D layout using Unity software, which can be useful for the developer. When designing levels, it can be beneficial to use references from other games to create a layout in a 2D view. Alternatively, a 3D layout can be created using Unity software, which allows for greater visualisation and interactivity for the developer. This process involves designing the environment, creating assets, placing objects, and considering the gameplay mechanics. A well-designed level can enhance the overall player experience and contribute to the success of a game.

  • Game Design Documentation

    The key process to kickstart game development is to create comprehensive documentation that encompasses various critical elements, including gameplay, level design, game mechanics, UI-UX references, character design, and an engaging storyline. As a Designer, it is crucial to ensure that the documentation is well-crafted, as it sets the foundation for a successful outcome in game development.
